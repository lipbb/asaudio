panproperty 
pan:Number  [read-write]

The left-to-right panning of the audio item, ranging from -1 (full pan left) to 1 (full pan right). A value of 0 represents no panning (balanced center between right and left). This value can be different from the "real" pan if the parent's item pan is different from 0.

volumeUnitsproperty 
volumeUnits:Number  [read-only]

[TBD, DO NOT USE] return an approximation of the amplitude in Volume Units (VU) from 0 to 1. Whereas peak() gives instantaneous amplitude, VU is intentionally a "slow" measurement, averaging out peaks and troughs of short duration to reflect the perceived loudness of the material as explained on Wikipedia.

cookieWrite()method 
public function cookieWrite(cookieId:String):Boolean

Writes audio item parameters to a flash cookie (SharedObject). This can be useful when using a music player accross different pages. Parameters saved :

  • volume
  • pan
  • position / positionMs (only for Tracks and Playlists). You must then use the start() with _useStartTimeFromCookie set to true.
  • currentTrackIndex (only for Playlists)

Parameters
cookieId:String — you must provide an id to retrieve the cookie later, using the cookieGet method.

Returns
Boolean — a boolean value indicating if the creation of the cookie has been succesful or not.

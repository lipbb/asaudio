A Group object is a container for Tracks or other Groups (or Playlists). It allows you to control a bunch of audio items globally, using the same methods and properties as the Track object (as both implements IAudioItem).

Constructor detail
Group()constructor
public function Group(_items:* = null, _name:String = null)

Creates a new Group object.

Parameters
_items:* (default = null) — an object containing children you want to add to the group :
  • If you provide a single Track, Group or Playlist object, this object is added as the single child of the group. Use addChild to add other children the same way later.
  • If you provide an Array containing Track, Group or Playlist objects, these objects are added to the group. Use addChild to add each other child individually later.
  • If you provide a string containing the url of a playlist file, the playlist is loaded and children are generated from this playlist. The playlist must have an xspf or xml extension for the XSPF format, or a m3u extension for the M3U format. Use loadXSPF and loadM3U to add tracks the same way later.
  • If you provide an XML object, the latter is interpreted as raw XSPF data, and children are generated from the playlist's items. Use addTracksFromXSPF to add tracks the same way later.
  • If you provide an Array containing strings, each string is interpreted as a track url, and children are generated from these urls. Use addTracksFromUrls to add tracks the same way later.
 
_name:String (default = null) — provide a name if you want to use the Group's method getItemByName(). If you don't provide a name, the name will be created from the auto generated uid.

cookieWrite()method 
public function cookieWrite(cookieId:String):Boolean

Writes audio item parameters to a flash cookie (SharedObject). This can be useful when using a music player accross different pages. Parameters saved :

  • volume
  • pan
  • position / positionMs (only for Tracks and Playlists). You must then use the start() with _useStartTimeFromCookie set to true.
  • currentTrackIndex (only for Playlists)

Parameters
cookieId:String — you must provide an id to retrieve the cookie later, using the cookieGet method.

Returns
Boolean — a boolean value indicating if the creation of the cookie has been succesful or not.
